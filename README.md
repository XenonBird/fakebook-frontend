## Route map

```
/ -> Overview
/about -> About
/contact -> Contact
/signup -> Signup
/login -> Login

/app -> AppHome (only accessible to registered users)
/app/home -> Home
/app/posts -> Posts
/app/posts/:postId -> Post
/app/posts/:postId/comments -> Comments
/app/new-post -> NewPost
/app/profile -> Profile
/app/settings -> Settings
```
